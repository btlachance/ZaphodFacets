But this advertiser is not so trustworthy now. If you click on this page, an attempt was made to redirect you to a 'malicious' site. Try running this page with SpiderMonkey enabled, and then rerun with the Zaphod engine enabled.

With Zaphod w/ Faceted Values, we run all code using the dom.js DOM, but we can monitor mutation events before they hit the underlying host DOM.

Be sure that the modified version of Zaphod is installed and active. You should see a small, red mozilla logo at the bottom right of your browser.
