Zaphod Addon - Narcissus JavaScript for the web (BETA)

Due to its reliance on JS Proxies and dom.js, this addon works only with Firefox 6 and above.

ZaphodFacets shows how we can provide confidentiality using a technique known as faceted values. Essentially, the idea is to track both a public and confidential value for sensitive data. See the ZaphodFacets page on github for more details.
